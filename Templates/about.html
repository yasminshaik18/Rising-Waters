About FloodGuard AI

A machine learning powered flood prediction and early warning system

Project Objective

To build an accurate and accessible flood prediction system using machine learning that helps authorities issue early warnings and manage disaster response effectively.

Dataset

Trained on over 1 million records with 20 environmental and meteorological features including monsoon intensity, deforestation levels, urbanization scores and more.

Technology Stack

Python, Scikit-learn, XGBoost, Flask, HTML, CSS. The best performing model is saved and served through a Flask web application.

Deployment

Designed for deployment on IBM Cloud making it scalable and accessible from any location by disaster management authorities.

Model Performance Comparison

Model Accuracy Status
Decision Tree 64.03% Baseline

XGBoost 81.60% ✅ Best Model (Deployed)

How It Works

1. User enters 20 environmental and meteorological values into the form
2. The Flask app receives the input and passes it to the XGBoost model
3. The model analyses the data based on patterns learned from 1 million+ records
4. A prediction is returned — either FLOOD LIKELY or NO FLOOD
5. The result page displays the risk level along with recommended actions
